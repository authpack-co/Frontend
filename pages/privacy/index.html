Privacy Policy

Last updated: March 16, 2026

AuthPack values your privacy and is committed to protecting your information. This Privacy Policy explains how the AuthPack browser extension and platform handle data.

1. Information We Collect

AuthPack may store authentication session data locally in the user's browser to enable session restoration and access automation across supported services.

AuthPack does not collect or store user passwords.

Some information may be transmitted securely to AuthPack servers when required to enable extension functionality, such as:

  • Account identifiers
  • Package metadata
  • Device connection information

2. How We Use Information

AuthPack uses collected information only to:

  • Enable session restoration
  • Manage user packages
  • Maintain active device connections
  • Improve extension reliability and performance

AuthPack does not sell, rent, or share personal information with third parties for marketing purposes.

3. Local Data Storage

Sensitive session data is stored locally within the browser environment and may be encrypted to protect user security.

This information is used exclusively to perform session restoration requested by the user.

4. User Consent

By installing and using the AuthPack extension, users acknowledge and consent to the processing of data as described in this policy.

Users control when sessions are created, restored, or removed.

5. Third-Party Services

AuthPack interacts with third-party websites only when the user explicitly accesses those services through the extension.

AuthPack is not responsible for the privacy practices of third-party platforms.

6. Data Security

AuthPack implements technical safeguards to protect stored information and prevent unauthorized access.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page with a revised date.
