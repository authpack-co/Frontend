Getting Started

Bem-vindo à documentação oficial do AuthPack! 🎉

O AuthPack é uma extensão para navegador que permite criar, organizar e compartilhar pacotes de autenticação de forma simples e segura.

Neste guia inicial você encontrará instruções básicas para começar a usar o AuthPack, incluindo como instalar a extensão e as dúvidas mais comuns.

➡️ Continue para a seção Instalação para aprender como configurar o AuthPack no seu navegador.
