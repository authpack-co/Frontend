FAQ

Quais navegadores são suportados?

O AuthPack funciona em navegadores baseados no Chromium, como Google Chrome, Microsoft Edge e Brave.

Preciso pagar para usar o AuthPack?

Não. O AuthPack é gratuito nesta fase inicial de desenvolvimento.

Por que preciso instalar via Developer Mode?

O AuthPack não está disponível na Chrome Web Store por questões estratégicas de segurança e distribuição. Mas a instalação via Developer Mode é simples e segura.

Onde posso obter suporte?

Você pode entrar em contato pelo nosso site oficial: authpack.co
